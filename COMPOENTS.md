# Fyarn UI Components

Lista de todos os componentes disponíveis no **Fyarn UI** para o comando `fyarn add`.

## Como usar

```bash
# Adicionar um componente padrão
fyarn add <nome_do_componente>

# Adicionar com uma variante específica
fyarn add <nome_do_componente> --variant <nome_da_variante>

# Adicionar múltiplos componentes de uma vez
fyarn add button card input
```

## Autenticação (Login & Signup)

Estes componentes possuem variantes de design:

### Login
- `fyarn add login --variant minimal`
- `fyarn add login --variant card`
- `fyarn add login --variant split`
- `fyarn add login --variant immersive`

### Signup
- `fyarn add signup --variant minimal`
- `fyarn add signup --variant card`
- `fyarn add signup --variant split`
- `fyarn add signup --variant immersive`
