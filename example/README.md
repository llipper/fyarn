# Fyarn UI Example

This directory contains a basic example of how to integrate the **Fyarn Design System** into your Flutter application.

## How to use Fyarn

1. **Activate the CLI**:
   ```bash
   dart pub global activate fyarn
   ```

2. **Initialize the Engine**:
   Run the following in your project root:
   ```bash
   fyarn init
   ```

3. **Add Components**:
   ```bash
   fyarn add button card dialog
   ```

4. **Wrap your App**:
   See the `main.dart` in this folder for the basic widget structure.

For more information, check the [root README](../README.md).
